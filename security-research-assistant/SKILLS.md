# Skills Reference

## Domain: Industrial Control System Security Research

### Key Concepts
- **Tear-down analysis**: Systematic disassembly and characterisation of hardware/software components
- **ICS (Industrial Control Systems)**: SCADA, PLCs, HMIs, field devices
- **Common interfaces**: SPI, I2C, UART, GPIO, Ethernet, RS-485
- **Common protocols**: Modbus, OPC-UA, MQTT, BACnet, DNP3, EtherNet/IP
- **Component types**: Processors, memory (Flash, EEPROM, SRAM), sensors, filters, fuses, regulators

### Source Quality Tiers
- **Tier 1**: Manufacturer documentation, official datasheets
- **Tier 2**: Academic papers, industry standards (IEEE, IEC)
- **Tier 3**: Trusted technical forums, known expert blogs
- **Tier 4**: General forum posts, unverified sources

### File Types Supported
- Documents: PDF, DOCX, TXT, MD, HTML
- Images: PNG, JPG, TIFF (OCR via Tesseract)
- Code: .c, .h, .py, .asm, .bin
- Data: CSV, XLSX
- Schematics: PDF-based wiring diagrams and circuit schematics
